# Inspira MVP

MVP Sprint 2 de Inspira, una plataforma educativa para conectar estudiantes secundarios con experiencias profesionales elegidas por interes.

## Que incluye

- Landing page
- Catalogo de experiencias
- Detalle de experiencia
- Flujo de inscripcion
- Dashboard estudiante
- Dashboard institucion
- Dashboard empresa
- Demanda vocacional agrupada
- Formularios para crear experiencias e intereses detectados

## Como correrlo

Con Node.js instalado:

```bash
node server.cjs
```

Abrir:

```text
http://localhost:4173
```

Tambien se puede abrir `index.html` directamente en el navegador, aunque se recomienda usar el servidor local.
